## Project Name:
 - KeenKeeper

## Description:
 - KeenKeeper is a responsive and user-friendly web application designed to showcase digital products and services.
 - It includes structured sections like navbar, banner, product cards, pricing, workflow, footer to help users easily explore and understand the platform.
- This project focuses on clean design, smooth layout, and reusable components.

## Technologies Used
- React.js  
- React Router
- Tailwind CSS, DaisyUI 
- JavaScript (ES6+)
- Rechart  
- React-Toastify 
- React Icon 
- JSON (for All Friends) 

## Features:
- Dynamic UI Sections: Includes Navbar, Products and Pricing component.
- Reusable Components: Clean and modular component structure for scalability.
- Responsive Design: Fully optimized for mobile, tablet, and desktop devices.
